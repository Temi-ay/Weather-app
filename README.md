# Weather App
## 📌Description
This is a simple lookup weather app. enter a city name and get the current temperature pulled from OpenWeatherMap Api
## ✈️ Features
-✅Search current weather by city name
-✅diplay humidity and temperature based on OpenWeatherMap
## Built with
- HTML
- CSS
- JAVASCRIPT
- [OpenWeatherMap Current Weather API](https://openweathermap.org/current)
